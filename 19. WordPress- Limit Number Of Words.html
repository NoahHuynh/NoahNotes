WordPress: Limit Number Of Words
Use Excerpt to limit and set number of words to display from blog.  Add code to Function.php 

// customized excerpt word count length
function custom_excerpt_length(){
return 60;
}

The number is the # of words you wish to display.

Next goto content.php and change/add <?php echo the_content();?>
to <?php echo the_excerpt();?>
