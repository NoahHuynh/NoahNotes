Style Everything except for:

#main .post:nth-of-type(3){
padding-bottom: 0px 0px !important;
}

This is a way to apply everything except for the first item.  So the first item will have padding-top:0px 
#main .post:nth-of-type(1){
	padding-top: 0px;
}

Another way to do things is 

div ul:not(:first-child){
background-color: black;
 }
