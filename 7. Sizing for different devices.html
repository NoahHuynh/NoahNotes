Sizing for different devices

There’s max which cannot exceed and there’s min which has to be a minimum of screen size:

@media (min-width:$screen-sm-min) {
position: absolute;
padding: 0px;
margin-top:4px;
right:0;
text-align: left;
a{
padding: 0px;
display: block;
margin-bottom: 20px;
}
}

@media (min-width: $screen-lg-min){
margin-top:-4px;
a{
padding: 8px 0;
display: inline-block;
}
}
}
